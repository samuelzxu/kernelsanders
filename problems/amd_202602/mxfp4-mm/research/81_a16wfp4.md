# #81 gemm_a16wfp4 — AMD's official fused bf16→fp4 + GEMM

## Key Discovery
`gemm_a16wfp4` is AMD's official kernel that:
1. Takes bf16 A directly (no separate quantization needed!)
2. Quantizes A to MXFP4 on-the-fly inside the GEMM kernel
3. Computes fp4×fp4 dot_scaled with the quantized A
4. Has proper XCD remapping, split-K support, etc.

This is exactly our custom fused kernel but AMD's production version.
It should work for ALL M sizes (not just M<=16 like our custom fused kernel).

## Expected improvement
- Eliminates dynamic_mxfp4_quant(A) for ALL shapes (~3µs saved per call)
- Single kernel launch: bf16 A + fp4 B + scale → bf16 C
- Properly optimized by AMD engineers

## Risk
- Correctness: on-the-fly quantization should match fp4×fp4 reference
- The kernel might have different config requirements (GEMM-A16WFP4 configs)
- May not support all our shapes (e.g., small M with split-K)
